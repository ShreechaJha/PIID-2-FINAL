# 📊 Sales Data Analysis — 2019

## 📝 Overview
This project is an end-to-end Exploratory Data Analysis (EDA) on 12 months of US retail sales data (approximately 186,000 orders). The goal of this analysis is to extract actionable business insights from real-world sales data, answering key business questions using Python.

## 🎯 Business Questions Answered
The analysis addresses the following 5 critical business questions:
1. **Which month had the best sales?** How much was earned that month?
2. **Which city generated the highest revenue?**
3. **What time should we display advertisements to maximize likelihood of customer's buying product?**
4. **What products are most often sold together?**
5. **What product sold the most? Why do you think it sold the most?**

## 🛠️ Technology Stack
- **Language:** Python 3.x
- **Libraries:**
  - `pandas` - For data cleaning and manipulation
  - `matplotlib` - For data visualization
  - `numpy` - For statistical operations

## 🚀 Project Structure
The Jupyter Notebook is divided into two main parts:
- **Part 1 — Setup:** Import libraries, merge 12 monthly CSV files into one DataFrame, clean the data (handling NaNs, converting data types), and feature engineering (adding useful columns).
- **Part 2 — Analysis:** Step-by-step statistical analysis and data visualization to answer the core business questions, culminating in key insights and business recommendations.

## 💻 How to Run
1. Clone the repository.
2. Navigate to the project directory and install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook DATA_ANALYSIS_PROJECT_FINAL_GITHUB.ipynb
   ```
4. Run the cells sequentially to view the data processing, visualizations, and insights.
