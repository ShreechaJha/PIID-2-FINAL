# PIID Project (Personally Identifiable Information Detection)

This project implements a multi-agent system to process, analyze, evaluate, and govern Personally Identifiable Information (PII) detection and risk mitigation.

## Directory Structure

- `DATASETS/` - Benchmark datasets (frozen).
- `MODELS/` - ML models (e.g., RoBERTa classifier, Logistic Regression).
- `AGENTS/` - Agent implementations (Preprocessing, Vision, Prompt, Risk, Governance, Decision).
- `RULES/` - System and compliance rules.
- `API/` - FastAPI/Flask application.
- `NOTEBOOKS/` - Jupyter Notebooks for EDA, training, and evaluation.
- `RESULTS/` - Outputs, metrics, and evaluation results.
- `PAPER/` - Academic/technical paper documents.
